# esg-financial-returns
Description (under 350 characters):  Empirical analysis of ESG performance and financial returns using CRSP, Compustat, and Refinitiv data. Panel of 18K+ U.S. firm-years (2013–2023). Covers multi-database merging, OLS/panel regressions with fixed effects, and industry-level double materiality classification. Python · pandas · statsmodels · WRDS
